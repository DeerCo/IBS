Quick Start

1. Requirements

Before proceeding, you need to have the latest stable node.js
Recommended environment:
  • node js 14+
  • npm js 6+

2. Install

Open package folder and install its dependencies. We recommanded yarn or npm.
1) Install with npm:
                            
cd project-folder
npm install
2) Install with yarn:
                            
cd project-folder
yarn install

3. Start

Once npm install is done now you an run the app.
                            
npm start or yarn start
This command will start a local webserver http://localhost:3000:
                            
Compiled successfully!

You can now view main in the browser.

  Local:            http://localhost:3000
  On Your Network:  http://192.168.43.100:3000

Note that the development build is not optimized.
To create a production build, use npm run build.

3.1 Start with NextJs

We have created Starterkit with Nextjs. Once npm install is done now you an run the app.
                            
npm run dev

4. Build / Deployment

For live your app you have add your live url folder path to package.json
                            
"homepage": "/",
After adding url run below command for build a app.
                            
npm run build or yarn build
Finally, Your app is ready to be deployed.🥳
